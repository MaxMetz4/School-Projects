About the Game

Why Was This Game Made?

This game was created to help people who are mentally challenged or for someone who wants to challenge their memory skills. The ability of the human mind can be expanded. Explore the lengths at which your mind can be expanded.


Lists of Illnesses The Memory Game can Help With

  • Amnesia,
  • Alzheimer's Disease,
  • Post-Traumatic Amnesia,
  • False Memory

The Memory Game can be used to help treat or train the brain to remember pictures of animals. The highest score means that the person has reached the optimal level of memorization. Score can vary due to randomization.
